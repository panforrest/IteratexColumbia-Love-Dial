## üé¨ Pre-Demo Checklist

### Before You Present:

- [ ] **Extension Loaded**: Make sure Love-Dial is loaded in Chrome
- [ ] **API Keys Set**: Verify ElevenLabs and Anthropic keys are working
- [ ] **Test Profile Ready**: Have 2-3 LinkedIn profiles ready to demo
- [ ] **Audio Working**: Test that voice preview plays correctly
- [ ] **Score Generation**: Verify romantic score generates successfully
- [ ] **Backup Plan**: Have screenshots/video if live demo fails
- [ ] **Browser Console**: Know how to show console logs if needed

### Demo Flow Practice:

1. Navigate to LinkedIn profile
2. Click pink "Call" button
3. Show modal opening
4. Click "Generate Voice Preview" ‚Üí Play audio
5. Click "Get Romantic Score" ‚Üí Show score and insights
6. Close modal

**Practice this flow 3-5 times before presenting!**

## üé§ Presentation Best Practices

### Delivery Tips:

1. **Energy**: Start with enthusiasm, maintain it throughout
2. **Pacing**: 3 minutes is tight - practice timing
3. **Eye Contact**: Look at judges, not just the screen
4. **Confidence**: You built this - own it!
5. **Transitions**: Smooth flow between sections

### Visual Aids:

- **Screen Share**: Show the extension working live
- **Backup Slides**: Have screenshots ready (just in case)
- **Console Logs**: Can show technical details if asked
- **GitHub Link**: Have repository URL ready to share

### Handling Questions:

- **Technical Questions**: Reference Version 2 script
- **Use Case Questions**: Reference Version 1 script  
- **Impact Questions**: Reference Version 3 script
- **"What if it breaks?":** Have backup demo ready
